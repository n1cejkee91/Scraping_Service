# Scraping_Service
https://vacancy-scraping.herokuapp.com/

Сайт на Django 3 "Сервис по сбору данных".

Cервис собирает данные о  вакансиях с сайтов по поиску работы и рассылает их подписчикам. Подписчики сервиса могут зарегистрироваться, выбирать город и язык программирования. Раз в сутки на основе предпочтений пользователей происходит рассылка вакансий на их почту.
